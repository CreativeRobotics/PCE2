PCExp1.pde - Java/Processing application
To use this software download the latest version of Processing from www.processing.org and follow the installation instructions.

Copy the Application Software folder to the processing Sketchbook folder and open the PCExp1.pde file to view and edit the code.

The application requires the controlP5 GUI library for Processing.

Install using the Library Manager:
In the processing application select: Sketch->Import Library...->Add Library...
Use the search box to find ControlP5
Select ControlP5 and click install.

To install manually:
This is in the libraries directory and needs to be copied to your local processing libraries directory. Processing->Sketchbook->Libraries->controlP5
